My Projects

POSN Camp 1 – Computer Science (Academic Olympiad Program)

Selected participant in the POSN Camp 1 (Computer Science), a national academic olympiad training program focused on advanced programming and algorithmic problem-solving.

During the camp, I studied C++ programming in depth, including data structures, algorithm design, time complexity analysis, and competitive programming techniques.

This experience strengthened my logical thinking, problem-solving skills, and ability to write efficient and optimized code under time constraints.

Fully Integrated Automated Seedling Cultivation System

This project focuses on designing and implementing an automated seedling cultivation system that integrates robotics, data communication, and intelligent control mechanisms to improve agricultural productivity.

Through this project, I developed technical expertise in real-time data transmission between microcontroller boards, user data handling and system reliability, as well as programming a robotic arm to execute accurate movements using a coordinate-based control framework.

Hydrogen Synthesis System for Alternative Energy Research

This project involves the development of a hydrogen synthesis system designed to explore alternative clean energy solutions and potential industrial applications.

The system produces hydrogen gas from water using an electrolysis process, where electrical energy is applied to split water molecules into hydrogen and oxygen.

Through this project, I gained a deeper understanding of electrochemical reactions, renewable energy technologies, and sustainable engineering design. I also developed practical skills in system construction, experimental testing, safety control, and performance evaluation of energy systems.

Machine Learning Model for Global Risk Analysis

This project focuses on developing a machine learning model to analyze and predict global risk factors using large-scale datasets.

The system evaluates multiple variables such as environmental indicators, economic trends, and social stability metrics to identify patterns and assess potential global risks.

Through this project, I gained experience in data preprocessing, feature engineering, model training and evaluation, as well as interpreting predictive analytics results for real-world decision-making applications.
